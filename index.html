CSS Selectors and Specificity

Selector Symbol Description Examples Specificity
Universal Selector * selects all elements within the html code. * {} least
Type Selector type selects the type of the html element that you would like to style. body {}, h1 {}, header {} 0
Descendant Selector space selects all the descendant elements of a parent element parent descendant {} 1
Parent/Child Selector > selects the first succeeding child of the parent element parent>child {} 2
Pseudo Class Selector :keyword keyword added to a selector that specifies a special state of the selected element(s). :focus {}, :visited {}, :hover {}, etc. 3
Class Selector . selects the value of the class attribute of an element. .className {} 4
Attribute Selector [attr] matches elements based on the presence or value of a given attribute. element[attrName] {} 5
ID Selector # selects the value of the ID attribute of an element. #idName {} 6
